# MACHINE LEARNING PROJECT
this project focus on inplementing machine learning algorithums to solve real-world data problems .It includes data processing , exploratory data anaylysis (EDA),
model training, and evaluation.

#### FEATURES
** Data Cleaning : ** Handling Missing Values and outliers.
** Features Engineering : ** Selecting the most relevant features for the model.
** Model Selection : ** Comparing different algorithum (e.g.., Regression , Classification).
** Visualization:- ** Detailed charts and graphs for better insights.

#### TECH STACK
** Lnguage:** Python
** libraries :- ** Pandas , Numpy , Scikit-learn , MAatplotlib , Seaborn

### HOW TO USE
